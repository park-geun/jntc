隐私政策

JNTC(以下称为"公司')为了保护用户的个人信息，遵守「关于促进信息通信网的使用和保护信息等的法律」、「个人信息保护法」以及关于个人信息保护的规定和指导方针。

公司通过隐私政策告知用户个人信息将以何种方式用于何种用途以及为保护个人信息采取了何种措施。


1. 个人信息的处理(收集、使用)目的

公司把用户的个人信息用于如下目的，处理的个人信息除了下面的使用目的外不会用于其他任何用途。若变更使用目的，则公司将事先征得同意。

  • - 对公司、产品、IR等的咨询及提供回复等其他服务
2. 个人信息的处理、使用和保存期限

公司只在法令规定的个人信息处理、使用、保存期限或收集个人信息时征得同意的个人信息处理、使用、保存期限内处理个人信息，在达到收集和使用个人信息的目的后立即销毁。

个人信息为了用为处理咨询事项的依据材料将保存一年。只是，若需要按照其他法令规定保存个人信息，则可以不销毁继续保存。

3. 信息主体的权利、义务以及其行使方法

个人信息的信息主体随时可以要求查询、修改登记的个人信息并取消对个人信息收集和使用的同意。只要个人信息的信息主体通过书面、 电话或电子邮件联系公司的个人信息管理部门，公司将立即按照信息主体的要求采取相应的措施。
若信息主体对错误的个人信息要求修改，则直到修改完为止该个人信息将不再使用或提供。

14周岁以下儿童由其法定代理人拥有查询和修改个人信息的权利及取消对个人信息收集和使用的同意的权利。

对按照信息主体或其法定代理人的要求注销或删除的个人信息，公司按照隐私政策的规定进行处理，不会为其他用途查阅或使用。

4. 处理的个人信息项目

公司收集如下个人信息。

  • - 个人信息收集项目：姓名、电子邮箱地址、联系方式(电话或手机号码)
5. 个人信息的销毁

当达到收集的个人信息的收集和使用目的或个人信息的保存期限到期时，公司除了用户同意或按照服务条款、相关法令需要继续保存的情况外，将立即销毁该个人信息。

  • (销毁方法)
  • - 记录个人信息的印刷品、书面材料等：粉碎或焚烧
  • - 电子文件：以无法还原的方法永久彻底删除
6. 个人信息自动收集装置的安装和运营及拒绝 公司不使用可以储存和随时调用信息主体的使用信息的"cookie'。

7. 个人信息安全措施

公司在处理用户个人信息时，为确保信息的安全采取如下措施，以防个人信息遗失、被盗、泄露、篡改以及毁坏。

  • (管理措施)
  • - 为了安全地管理个人信息，制定并实施内部管理计划。
  • - 以操作个人信息的人员为对象，每年实施一次以上定期培训和提高安全意识的活动。
  • - 最大限度的限制可以访问个人信息的人员。
  • (技术措施)
  • - 为了防止因黑客入侵或电脑病毒等个人信息泄露、毁坏，公司安装安全程序来阻止黑客等外部入侵。
  • - 信息主体的个人信息和密码将加密保存和管理，传输信息时也将通过其他安全功能安全地进行管理。
  • (物理措施)
  • - 运营机房和资料保管室等保护区来控制人员出入。
8. 员工个人信息的处理

公司对处理员工的个人信息也按照本隐私政策的规定进行。
